# Support

Product Wiki is an open-source agent harness.

For bugs, open an issue with:

- the repo type: new product or existing repo
- the agent used: Codex, Claude Code, or both
- the command or prompt that failed
- relevant output from `node scripts/product-wiki-check.mjs` and `node scripts/doctor.mjs`

For security issues, do not open a public issue.
Email hi@omarismail.com.
